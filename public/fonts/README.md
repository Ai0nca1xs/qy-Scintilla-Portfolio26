# Favorit 字檔放置處

站點文字棧為:`Favorit → Familjen Grotesk(免費回退)→ Helvetica`。
要啟用真正的 ABC Favorit(Dinamo 商業字體,需持有授權),把以下檔案放進本資料夾:

- `Favorit-Regular.woff2`(400)
- `Favorit-Medium.woff2`(500)
- `Favorit-Bold.woff2`(700)
- `Favorit-RegularItalic.woff2`(400 italic)

檔名需完全一致(對應 global.css 的 @font-face)。沒有這些檔案時自動用
Familjen Grotesk,不會壞版。字檔本身**不要**提交到公開 repo 以外散佈授權範圍。
